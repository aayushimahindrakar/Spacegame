SPACE GAME!

-I coded this fun game you can play by yourself or with a friend. The rules of the game are simple: DONT HIT THE ASTEROIDS!! They start coming faster the further you get.

-Coded in javascript played in html.

-Future improvements may include levels, power ups, game modes, improved graphics, etc.

-Made by me (new to this btw) with a little help from ai for debugging.

Want to install
-Clone the repository: git clone https://github.com/aayushimahindrakar/Spacegame.git
-Open the project folder.
-Run the game in your browser or local development environment.
